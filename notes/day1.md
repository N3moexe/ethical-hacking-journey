## Networking Notes

### What is an IP Address?
An IP address is like a house address for a computer on a network.

### What is a Port?
A port is like a door on a house that allows specific services to communicate.

### What is DNS?
DNS translates website names into IP addresses.

### What is HTTP?
HTTP is the language used by web browsers and websites to communicate.
